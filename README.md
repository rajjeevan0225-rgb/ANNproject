# ANN Mini Project

This project contains basic implementations of Artificial Neural Networks using Python.

## Projects

- **Perceptron**
  A simple model used for binary classification. Tested on basic input data.

- **Competitive Learning**
  An unsupervised model where neurons compete to represent input patterns and form clusters.

- **Feedforward Neural Network**
  A multi-layer network built using TensorFlow and trained on the Iris dataset.

## Tools
Python, NumPy, Matplotlib, Scikit-learn, TensorFlow

## Outcome
The models were able to learn patterns from the data and produce correct outputs.
